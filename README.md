# Roblox Manager (macOS)

A lightweight Roblox multi-instance manager for macOS that allows users to run multiple Roblox sessions with different accounts at the same time.

## ✨ Features

### 🧩 Multi Instance Support
Run multiple Roblox instances simultaneously without conflicts or session overlap.

### 👤 Multi Account Handling
Assign a different account to each instance for easy switching between profiles.

### 🌐 Browser Login
Log in through the official Roblox browser flow for secure authentication.

### 🍪 Cookie Login
Quickly sign in using stored `.ROBLOSECURITY` cookies for faster access without re-authentication.

---

## 🚀 Use Cases

- Managing multiple Roblox accounts at once
- Testing games with different users
- Running alternate profiles simultaneously
- Faster account switching for developers and power users

---

## ⚙️ Requirements

- macOS (Sequoia 15+ recommended)
- Internet connection
- Valid Roblox accounts

---

## 🔐 Security Note

This tool does not modify Roblox client files. However, using multiple accounts or automation tools may still be subject to Roblox's Terms of Service. Use responsibly.

Your Roblox cookies are encrypted and stored only on your local computer. They are never uploaded, shared, or sent to any external server, the encryption helps protect them from being easily read or exposed by other applications, including potential cookie loggers, in the event your files are accessed.
